# Free Stack

Recommended setup for now:

1. Vercel free deploy for the web app
2. Local-first storage in the browser
3. JSON backup export/import
4. GPT Project markdown export for reflection

Why this is good now:

- no monthly database bill
- simple personal workflow
- fast enough for one person
- easy to move to Supabase later

When to add Supabase later:

- you want live sync across phone and desktop
- you want login
- you want reliable shared multi-user storage
